# 💌 Love Proposal Page

A cute, interactive **love proposal web page** built with pure HTML, CSS & JavaScript — no frameworks needed! Send this link to your special someone and let the magic happen. 💕

---

## ✨ Features

- 💖 **Animated Proposal Screen** — Cute images displayed in a card grid
- 🏃 **Runaway "No" Button** — The No button escapes your cursor every time you try to click it (you can never say no! 😄)
- 🎊 **Thank You Screen** — Confetti explosion + sweet message when she says Yes!
- 🌸 **Floating Petals Animation** — Romantic floating emoji petals in the background
- 🛠️ **Builder Modal** — Customize names, headings, and messages to make it your own
- 📱 **Fully Responsive** — Works beautifully on mobile and desktop

## 🚀 How to Use

### Option 1 — Open Locally
Just open `index.html` in any web browser. No installation needed!

### Option 2 — Share via GitHub Pages
Visit the live link above and share it with your special someone 💌

## 🛠️ Customization

Click the **"✨ Create yours"** button on the Thank You page to open the Builder and customize:
- Your name & her name
- Proposal heading & tagline
- Thank-you page heading & message
